# WeakestLinkUI
Useable (but not great) early dev version of what will one day become Weakest Link UI app.

## Description

WeakestLinkUI is a user interface application for the Weakest Link game. This early development version provides basic functionality for gameplay, including player management, question presentation, and elimination mechanics. The application is actively being developed with planned improvements to UI/UX, performance, and feature completeness.

## Features

- Player management and tracking
- Question display system
- Round-based elimination
- Score calculation

## Status

This is pre-release software. Expect bugs and incomplete features.
