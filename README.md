# WeakestLinkUI
Useable (but not great) early dev version of what will one day become Weakest Link UI app.

## Description

WeakestLinkUI is a user interface application for the Weakest Link game. This early development version provides basic functionality for gameplay, including player management, question presentation, and elimination mechanics. The application is actively being developed with planned improvements to UI/UX, performance, and feature completeness.

## Installation

### Prerequisites
Before running WeakestLinkUI, you need to install the `customtkinter` library. If you don't have Python installed, download it from [python.org](https://www.python.org/downloads/).

### Installing customtkinter

1. Open Command Prompt (Windows) or Terminal (Mac/Linux)
2. Run the following command:
    ```
    pip install customtkinter
    ```
3. Wait for the installation to complete

Once installed, you can run the application.
